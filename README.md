# nyc-trip-planner

The New York Line — an interactive NYC trip planner (FIFA World Cup 26, 19–25 June 2026)

## Live site

**https://calebleungg.github.io/nyc-trip-planner/**

A single self-contained `index.html`: a timeline view, an editable drag-and-drop
planner (where you can add your own activities), and a map of every stop. The
timeline reflects whatever you arrange in the planner, and your changes are saved
in your browser.
